Projeto Brasileirão 2025: Análise de Dados com Power BI
Este é um projeto pessoal de análise de dados e business intelligence focado no Campeonato Brasileiro de 2025 (Brasileirão Série A). O objetivo é explorar e visualizar dados de forma interativa, transformando informações brutas em insights valiosos sobre o desempenho dos times e jogadores.

Objetivos do Projeto:
Coleta e Tratamento de Dados: Utilização de dados fornecidos em arquivos .csv para análise.
Criação de Dashboard Interativo: Desenvolvimento de um dashboard completo no Power BI para visualizar estatísticas de forma clara e intuitiva.
Análise de Desempenho: Apresentação de dados sobre a classificação do campeonato, artilheiros, líderes de assistência, e estatísticas de público.

Tecnologias Utilizadas:
Power BI: Ferramenta principal para a criação do dashboard e visualização de dados.
CSV: Formato dos arquivos de dados brutos.
Gemini AI: Utilizado para auxiliar na estruturação e na escrita do projeto.
PHP e Banco de Dados (Futuro): Próxima etapa do projeto, com o objetivo de automatizar a coleta de dados e tornar o dashboard dinâmico.

Estrutura do Repositório:
dados/: Contém todos os arquivos .csv utilizados na análise (Artilharia.csv, Assistências.csv, Jogos.csv, Público.csv, Tabela.csv).
dashboard/: Contém o arquivo do projeto no Power BI (.pbix) e capturas de tela do dashboard.
README.md: Este arquivo, com a descrição completa do projeto.

Visão Geral do Dashboard:
O dashboard oferece uma visão completa do campeonato, com as seguintes análises:
Classificação: Tabela detalhada com pontos, vitórias, empates, derrotas, gols e saldo de gols.
Artilharia e Assistências: Listas dos jogadores com maior número de gols e passes para gol.
Público: Análise de média de público e taxa de ocupação dos estádios.

Este projeto demonstra minhas habilidades em Power BI, Data Analytics e a capacidade de transformar uma paixão em um projeto profissional.
